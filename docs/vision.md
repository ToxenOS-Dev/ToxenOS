# ToxenOS Vision

ToxenOS is an experimental operating system focused on clarity,
control, and intentional design.

## Core Ideas
- A small, understandable kernel
- Tasks over processes
- Devices as first-class systems, not files
- No POSIX or Linux compatibility layers

## Initial Goals
- Boot to text output
- Single address space
- Cooperative tasks

## Long-Term Ideas
- Custom syscall model
- Graphical environment
- Strong separation between kernel and runtime
